# Unix_Project - Smart Mirror ü™û

Smart Mirror is a hidden screen placed behind a two-way mirror. Powered by a Raspberry Pi, the screen displays real-time information such as the date, the time, the weather, and a quote of the day, all visible through the mirror's surface.
***
### üéØ Project Goals 
The goal of this project is to construct a fully functional, open-source Smart Mirror using a Raspberry Pi. This project's main objective is to transform a standard mirror into a interactive digital system that provides essential, real-time informations visible at a glance in the mirror, making the morning routine more efficient and informative.

### üìù Requirements
- Raspberry Pi
- Linux kernel to build the script and interface 
- Displayable screen
- microSD card with Raspberry Pi Imager (download from https://www.raspberrypi.com/software/)
- Connectors cable (e.g. HDMI cable) to connect the Raspberri Pi to the screen.
- A two-side mirror glass or film
